# Hotel Management – Visual Planning Suite

This module is the visual brain of the hotel system. It lets staff see and manage the entire property the way they see it in real life—rooms, tables, and bookings—using simple drag-and-drop screens instead of complicated forms.

---

## 1. Visual Floor Plan Editor

### What It Is

A digital map of the hotel where every room, corridor, hall, and facility can be arranged just like on paper—only smarter.

### Key Ideas

* **Design Like Drawing**

  * Staff can create floors and place rooms visually.
  * Rooms can be resized, moved, and labeled.
  * Different areas like lobby, restaurant, pool, and parking can be marked.

* **Live Room Status**

  * Each room changes color based on what’s happening:

    * Available
    * Occupied
    * Reserved
    * Cleaning
    * Maintenance

* **One-Click Actions**

  * Click a room to:

    * Create a new booking
    * Check guest details
    * Mark cleaning or maintenance
    * Move a guest to another room

* **Smart Guidance**

  * The system warns if:

    * Two rooms overlap
    * A room number is repeated
    * Capacity doesn’t match the room type

* **Perfect for Non-Tech Users**

  * Works like arranging furniture on a tablet.
  * No technical knowledge required.

## 2. Table & Seating Plan Editor

### What It Is

A visual tool to manage restaurant tables, banquet seating, and event layouts.

### Key Ideas

* **Arrange Tables Visually**

  * Add round or square tables
  * Place chairs around them
  * Create zones like:

    * Indoor
    * Terrace
    * VIP
    * Family area

* **Live Table Status**

  * Free
  * Reserved
  * Occupied
  * Bill pending
  * Cleaning

* **Reservations Made Easy**

  * Staff can:

    * Select a table directly from the map
    * Choose time slots
    * Combine tables for big groups
    * See capacity instantly

* **Event Friendly**

  * Layouts for weddings, conferences, parties
  * Save multiple seating styles
  * Reuse plans for future events

## 3. Gantt-Style Booking Schedule

### What It Is

A timeline view that shows every room booking like colored bars on a calendar—simple, visual, and powerful.

### Different Views

* **Day View**

  * Hour-by-hour picture of arrivals and departures
  * Cleaning and maintenance slots visible

* **Week View**

  * Each room in a row
  * Each day in a column
  * Easy to see gaps and availability

* **Month View**

  * Big picture of occupancy
  * Spot busy periods instantly

* **Year View**

  * Long-term planning
  * Seasons and events overview

### What Staff Can Do

* Drag a booking to another room
* Extend or shorten a stay by pulling edges
* See conflicts before they happen
* Open guest details with one click

### Booking Cards Show

* Guest name
* Number of people
* Payment status
* Special services
* Source of booking

## How Everything Works Together

### Floor Plan + Schedule

* Click a room on the map → see its bookings
* Move a booking on timeline → room map updates
* Cleaning status appears everywhere

### Tables + Bookings

* A single event can reserve:

  * Rooms
  * Tables
  * Banquet halls
    at the same time.
